# 阿拉俱乐部

  "阿拉俱乐部"是一款线上预定唱吧和录音棚的应用。
  
  1.唱吧:
  唱吧分类量贩式和迷你两种唱吧类型;
  唱吧详细的地址和评分,以及唱吧的最低价格等数据。

  2.录音棚
  提供成都范围内的6家优质录音棚,口碑均不错的录音棚。
  
  3.预定
  唱吧和录音棚均有大包/中包/小包的包间,用户根据人数预定不同规格的包间;
  选择好的包间类型,再选择预约时间和联系人信息,即可下单成功。
  
  4.订单
  订单分为三大状态:已预约、已使用、已取消。
  
  5.我的
  个人信息修改、查看我的收藏人数和列表、我的足迹数量和列表、关于我们、应用设置等。
